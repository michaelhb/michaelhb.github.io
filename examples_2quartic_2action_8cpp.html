Detailed Description

Program that calculates action for a one-dimensional potential parameterized by \(E\) and \(\alpha\).

The potential is

\[ V(\phi) = E\left[\frac{-4 \alpha +3}{2} \phi^2 - \phi ^3 + \alpha \phi ^4\right]. \]

Usage:

./action <E> <alpha> <dim>
