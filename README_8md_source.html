1 BubbleProfiler
2 ==============
3 
4 BubbleProfiler is a C++ library for calculating the bounce solution
5 and Euclidean action associated with a cosmological phase transition.

10 Requirements
11 ------------
12 
13 The following dependencies are required to build BubbleProfiler:
14 
15  * [CMake] (version 2.8.12 or higher)
16  * C++11 compatible compiler (g++ >= 4.8.1 or clang++ >= 3.8 or icpc >= 15.0)
17  * [Boost] (version 1.53.0 or higher), specifically the components
18  - Boost.Program\_options
19  - Boost.Filesystem
20  - Boost.System
21  * [Eigen 3] (version 3.1.0 or higher)
22  * [GiNaC] (version 1.6.2 or higher)
23  * [GNU scientific library] (version 1.15 or higher)
24  * [NLopt] (version 2.4.1 or higher)

27 
28  * [Doxygen], for building the documentation

31 Building BubbleProfiler
32 -----------------------
33 
34 BubbleProfiler uses CMake to configure its build system. To build the
35 library and command line executable,
36 
37 1. Create a build directory in which to build the package:
38 
39  mkdir build
40 
41 2. Run CMake from this directory to configure the build system:
42 
43  cd build
44  cmake ..
45 
46 3. Compile the core library and command line executable:
47 
48  cmake --build .
49 
50 4. Optionally, build and run the unit tests:
51 
52  cmake --build . --target 'check'
53 
54 5. Optionally, build the example programs:
55 
56  cmake --build . --target 'examples'
57 
58 6. Optionally, build the package documentation:
59 
60  cmake --build . --target 'docs'
61 
62 7. Optionally, install the library and executables:
63 
64  cmake --build . --target 'install'
65 
66 After building the package, the build directory will contain the
67 subdirectories
68 
69  * `bin/`, containing the main command line executable and the
70  example programs, if compiled
71  * `docs/`, containing the package documentation, if generated
72  * `examples/`, containing example scripts, if generated
73  * `include/`, containing generated header files
74  * `lib/`, containing the BubbleProfiler library
75  * `test/`, containing test binaries and output, if compiled
76 
77 For more detailed instructions and help with the build and installation
78 process, refer to the accompanying INSTALL.md file.

80 Running BubbleProfiler
81 ----------------------
82 
83 BubbleProfiler consists of two core components, a command line executable
84 for standalone use and a library to be used in your own programs. The
85 executable `run_cmd_line_potential` calculates the action and, optionally,
86 the field profiles for the bounce solution, given a potential and set of
87 fields. For example, the bounce action for a potential involving two scalar
88 fields may be calculated using
89 
90  ./bin/run_cmd_line_potential \
91  --potential "(x^2 + y^2)*(1.8*(x - 1)^2 + 0.2*(y-1)^2 - 0.4)" \
92  --field x --field y --local-minimum 0 0
93 
94 A summary of the possible options for the program may be printed using
95 `./bin/run_cmd_line_potential --help`. Detailed information about
96 the program and its options may be found in the accompanying manual.
